# Copyparty Grid Video Player

Play any video in the copyparty grid view using just a single JavaScript file and a video.

## How to use

1. Copy `badapple.js` and a video into a folder on your copyparty server.
2. Click the Ï€-symbol below the file view.
3. Type `import("[path to where you are]/badapple.js")` and press enter.
4. Enter the path to the video you want to play.
5. Enter a zoom anmount. This can be any number above 0, you can omit the "%" if you'd like.
> [!NOTE]
> If the video lags to much, try do increase the zoom. This way less have to be created and updated.
6. Enter a name for the files, you can leave this empty if you want.
7. Done! :thumbsup:
